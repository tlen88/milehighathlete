# Mile High Athletics — Website Handbook (for Ty)

> The friendliest version of this handbook lives inside your portal: **milehighathlete.com/admin/help/** — bookmark it. You can also print that page to PDF (File → Print → Save as PDF) for a paper copy. This file is the same content in plain text.

## Welcome

Your website now has a private **Site Manager** — a login-protected portal where you change what the website says and shows, without touching code.

**You can update:** headlines, paragraphs, button text, your photo, program cards, reviews, the hyperbaric section, your Vagaro booking link, contact and social links, and how the site appears on Google.

**You can't accidentally break:** the design, colors, fonts, layout, animations, mobile behavior, or the booking system. Those are locked. If a change would break the layout (like a headline that's far too long), the editor warns you before you can save.

## Logging in

1. Go to **milehighathlete.com/admin/**
2. Click **"Log in to edit your website."**
3. Enter your email and password. First time? Open your invitation email, click **Accept the invite**, and choose a password.
4. Log out with the **Log out** button, top right.

**Forgot your password?** Click "Forgot password?" on the login window — a reset link is emailed to you. Check spam if it doesn't arrive.

## Editing text

Open a card → click into a field → type. Grey helper text under each field explains where it appears. The preview pane on the right updates as you type. Click **Save** — your change is now a **draft**, not live. Keep text short and punchy; fields warn you in red if text is too long.

## Editing images

Open the section → click the image field → **Choose an image → Upload** → pick a file → fill in the **Alt Text** (one-line description) → Save → Publish.

| Image | Shape | Size |
|---|---|---|
| Hero photo of you | Vertical (portrait) | ≈ 1350 × 1688 px (4:5) |
| Chamber photo | Horizontal | ≈ 800 × 600 px (4:3) |
| Benefits graphic | Wide | ≈ 1672 × 941 px (16:9) |
| Social share image | Horizontal | ≈ 1200 × 630 px |

Use JPG for photos, keep files under ~300 KB when you can, never stretch a small photo.

## Programs & pricing

- **Edit:** open Training Programs, click a program to expand it.
- **Add:** "Add Programs" button (three looks best on desktop).
- **Hide without deleting:** turn "Show On Website" off.
- **Reorder:** drag with the handle on the left.
- **Highlight one:** turn on "Highlight This Program" — one program only.
- **Pricing:** write prices exactly as you want them shown, e.g. "$150 per session". Double-check numbers before publishing.

## Preview → Save → Publish

1. **Save** stores your change as a **Draft** — the live site is untouched.
2. Set the draft's **Status** to **Ready**.
3. Click **Publish → Publish now.**
4. The site rebuilds itself. After a couple of minutes, refresh milehighathlete.com (Cmd+Shift+R / Ctrl+Shift+R) and confirm.

**If publishing fails:** nothing is lost — your change stays saved as a draft. Try again in a minute; if it keeps failing, contact support. The live site is never broken by a failed publish.

## Undoing mistakes

- **Before saving:** just retype or leave the page.
- **Saved, not published:** fix the draft or delete it — the live site was never touched.
- **Already published:** edit back and publish again, or ask support — **every published version is kept forever** and can be restored exactly.

## SEO in plain English

- **Browser Tab / Google Title** — the blue headline in Google results. Include "Denver" + what you do.
- **Google Description** — the blurb under it. Make it an invitation.
- **Alt text** — one-line photo description; Google reads it.
- **Social Share Image** — the picture people see when your link is texted or posted.

## Safety

Safe any time: text, photos, programs, reviews, links, SEO fields.
Ask first: the "Website Address (canonical)" field, the wellness disclaimer, scheduler height.
Not editable on purpose: design, colors, fonts, layout, the booking system itself.

## Troubleshooting

| Problem | Fix |
|---|---|
| Forgot password | Login window → "Forgot password?" → check email/spam |
| Invite link expired | Ask support to re-send it |
| Change not showing | Did you Publish (not just Save)? Wait 2–3 min, hard-refresh |
| Blurry image | Upload a larger file (see sizes above) |
| Publish button greyed out | Set Status to "Ready" first |
| Wrong link | Field must start with `https://` — paste the full address |
| Looks broken on phone only | Almost always cache — try a private window |
